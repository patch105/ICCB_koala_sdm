Description

Abstract / Description:
This topographic dataset contains point features representing the centre of named towns and cities.
Purpose:
This Queensland wide dataset provides generalised information for topographical mapping purposes. To provide the central location of named towns and cities for the State of Queensland. The primary objective of this dataset is to provide a cartographic portrayal of Queensland's named features and should be interpreted and utilised correctly. It is suitable for use in land administration, emergency management, topographic mapping and in the production of web based mapping applications.
Additional Information:
The location of the feature is generally placed in the centre of the roadway adjacent to the main post office. If this is inappropriate, then in the centre of the roadway in the most central part of the community, generally the main commercial area. Names have been sourced from the Queensland Place Names Database.
Feature Types:
Populated centre - A point representing the centre of a named urban settlement (town or city).

Attributes:
FEATURE TYPE - Unique feature type that identifies the topographic feature e.g. Populated centre.
NAME - The name of the feature.
ALTERNATE NAME - Alternate name of the feature (if one exists)
QLD PNDB ID - Queensland Place Names Database unique identifier
ATTRIBUTE SOURCE - The source map, or data where the name of the feature was obtained from.
ATTRIBUTE DATE - Date of the source map, or data where the name of the feature was obtained from.
ADDITIONAL NAME - Other names of the feature if not recorded in the Queensland Place Names Database. Maybe commonly known by other names or have different spelling. Multiple names can be shown, separated by a comma.
ADD NAME SOURCE - The source map or data that the additional names were obtained from. Multiple sources can be shown, separated by a comma. If multiple sources are shown, the written sequence will be the same as the written sequence of the names themselves.
FEATURE SOURCE - The imagery, map or data that the spatial location of the feature was obtained from.
FEATURE DATE - Date of the source imagery, map, or data that the spatial location of the feature was obtained from.
OPERATIONAL STATUS - The operation status of the building. Options: Abandoned, Historic, Operational, Non Operational, Ruin
POPULATION - The population (number of persons) indicated by the ABS Census figures that exist for this locality. This field will no longer be updated as of the 01/03/2016 due to changes in the source data from ABS data.
POPULATION DATE - The date of the ABS census data
LOCAL GOVERNMENT - The name of the local government
PFI - A Persistent Feature Identifier (PFI) is generated for each feature at the point of creation in the database. The value of the PFI will stay with the feature through all changes to the feature (both spatial and non-spatial) until the feature is retired. (See the Persistent and Unique Feature Identifiers under the Data Model header for more information)
UFI - A Unique Feature Identifier (UFI) is generated for each new feature at the point of creation in the database (at this point the PFI and UFI will be the same). The value of the UFI will stay with the feature through all changes to the feature (both spatial and non-spatial) unless the feature is split into multiple other parts. If the feature is split, the separate parts will retain the original PFI but new UFI’s will be generated for the split parts. (See the Persistent and Unique Feature Identifiers under the Data Model header for more information)
UPPER SCALE - The upper scale for which the feature should be considered suitable
TEXT NOTE - For mapping purposes. Additional annotation that can be shown on a map.
ADD INFORMATION - A comment field (additional information) This dataset is stored in an ESRI Geodatabase. Some column names may be shortened if downloaded in the ESRI shape format.

History:

Lineage statement:
This dataset is GDA2020 compatible. The source data, and therefore the captured features, are georeferenced as GDA94 (horizontal only). The horizontal accuracy of the geo-referencing and/or data collection method for this example is greater than the datum offset between GDA94 and GDA2020 (1.8 meters). The resulting dataset has been nominated as a low-accuracy GDA2020 dataset. As this data has not been directly captured in GDA2020, nor transformed to GDA2020, the resulting data is 'GDA2020 Compatible' not 'GDA2020 Compliant'.
Data is extracted as a coordinate from the Queensland Place Names Database (PNDB) that has a Place Type classification of either City, Town or Township. The coordinates of this data is to the nearest minute. These points are then moved visually against imagery to a more appropriate location. The location of the feature is generally placed in the centre of the roadway adjacent to the main post office. If this is inappropriate, then in the centre of the roadway in the most central part of the community, generally the main commercial area.
The population figures have been arrived at by extraction of data from the Australian Bureau of Statistics (ABS) census. The population figures have been arrived at by extraction of data from the Australian Bureau of Statistics (ABS) census. A date within the data states the year of the ABS census. This field will no longer be updated as of the 01/03/2016 due to changes in the source data from ABS data.
